# 2. DESCRIPTION DE L’ONG

ONG Vahatra est une organisation malgache spécialisée dans :

- la microfinance sociale
- le microcrédit rural
- l’accompagnement des familles vulnérables
- le développement agricole
- l’élevage
- la mutuelle santé
- l’inclusion financière
- la résilience économique rurale

L’organisation intervient principalement dans :
- Vakinankaratra
- Itasy
- régions rurales de Madagascar
